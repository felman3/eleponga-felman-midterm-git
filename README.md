## Midterm Examination Reflection

**1. What is Git used for?**

Git is used to track changes in project files. It saves the work step by step, so it is easy to go back to an older version if something goes wrong. It works like a save point for a project.

**2. What is the difference between Git and GitHub?**

Git is a tool installed on the computer that tracks changes in a project, while GitHub is a website where the Git project can be uploaded and stored online. Git works even without internet, while GitHub needs internet because it is an online platform.

**3. Why is branching useful?**

Branching allows new changes to be tested without touching the main version of the project. If something goes wrong in the branch, the main branch stays saf and when the changes are correct, the branch can be merged back into main.
